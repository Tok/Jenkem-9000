Jenkem-9000

Web Application to convert images into IRC-style HTML&CSS

Download Download fork Jenkem-9000 on GitHub

Welcome to Jenkem-9000 on GitHub.

Jenkem-9000 is a Web-Application to convert Images into IRC-style HTML&CSS and generate colored output for IRC.

This application is running at: https://jenkem-9000.rhcloud.com/

Info page: https://jenkem-9000.rhcloud.com/#info/

Direct link to image conversion (just add image url after #main/): https://jenkem-9000.rhcloud.com/#main/http://upload.wikimedia.org/wikipedia/commons/0/03/RGB_Colorcube_Corner_White.png

The source code can be seen at: https://github.com/Tok/Jenkem-9000
